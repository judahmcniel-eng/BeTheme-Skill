# BeTheme Cinematic Scroll Heroes

A Codex skill for building, troubleshooting, and optimizing cinematic scroll hero sections in WordPress BeTheme/BeBuilder.

The actual skill instructions live in [`SKILL.md`](./SKILL.md).

## What It Helps With

- BeTheme Global Sections
- BeBuilder cinematic heroes
- Parallax hero layouts
- Sprite sheet scroll effects
- JPG/WebP frame sequence heroes
- Mobile fallbacks
- Performance checks
- `max_input_vars` save issues
- Safe BeBuilder Code element snippets
