# BeTheme Cinematic Scroll Heroes

A Codex skill for building, troubleshooting, and optimizing cinematic scroll hero sections in WordPress BeTheme/BeBuilder.

The actual skill instructions live in [`SKILL.md`](./SKILL.md).

**One-click ready for Claude Projects and Claude Code.**

## How To Add This Skill To Claude

Best way, if you are using Claude Projects:

1. Open [claude.ai](https://claude.ai) and go to any Project.
2. Go to **Knowledge**.
3. Click **+** or **Add content**.
4. Choose **GitHub**.
5. Paste this repo URL:

```text
https://github.com/judahmcniel-eng/BeTheme-Skill
```

6. Add [`SKILL.md`](./SKILL.md) first.
7. If Claude lets you add more files, also add the files inside [`references/`](./references).

Fast fallback:

1. Open [`SKILL.md`](./SKILL.md).
2. Copy the full file.
3. Paste it into Claude Project Knowledge.

## What It Helps With

- BeTheme Global Sections
- BeBuilder cinematic heroes
- Parallax hero layouts
- Sprite sheet scroll effects
- JPG/WebP frame sequence heroes
- Mobile fallbacks
- Performance checks
- `max_input_vars` save issues
- Safe BeBuilder Code element snippets
