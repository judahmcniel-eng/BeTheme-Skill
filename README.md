# BeTheme Cinematic Scroll Heroes – Complete Mastery Skill

**One-click ready for Claude Projects, Claude Code, and AI-assisted WordPress/BeTheme work.**

## Purpose

This repository gives Claude, Novamira, or any AI coding/design assistant a compact, production-ready knowledge base for building cinematic scroll heroes in BeTheme’s BeBuilder using native elements, Global Sections, sprite sheets, JPG sequences, and safe custom code.

## How to Add This Skill to Claude

### Best way

1. Open Claude.
2. Go to a Project.
3. Knowledge → Add → GitHub.
4. Paste this repository URL.
5. Add `README.md` and, if available, `skill/betheme-cinematic-scroll-heroes.md`.

### Alternative

Open `skill/betheme-cinematic-scroll-heroes.md`, click **Raw**, copy everything, and paste it into Claude Project Knowledge.

Fast Start: Build a Cinematic Global Hero
Create a new Global Section in BeTheme.
Set the Section to full-screen / 100vh.
Add a sprite sheet, JPG sequence, static image, or video as the background layer.
Add Overlay and Shape Divider for cinematic depth.
Add foreground content: Fancy Heading, supporting text, and Button.
Add a Code element only when the native BeBuilder tools are not enough.
Save the section as a reusable Global Section.
Drop it anywhere on the site.
Test mobile, performance, and max_input_vars before publishing.
Final Pro Tips
Use WebP everywhere possible.
Use a CDN for sprite sheets and large background assets.
Use Global Styles + Presets for consistency.
Use Layer Navigator constantly when working with absolute layers.
Keep cinematic sections restrained: 2–4 moving layers is usually enough.
Always test with GTmetrix, Query Monitor, and real mobile devices.
If BeBuilder or Theme Options refuse to save, increase max_input_vars to 10000+.
